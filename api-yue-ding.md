# API 约定

Elasticsearch 的 REST API 通过  [HTTP](https://www.elastic.co/guide/en/elasticsearch/reference/2.2/modules-http.html "HTTP")协议和 JSON 数据格式供用户使用。

除非另有说明，本章的约定适用于所有的 API
- 多重索引
- 索引名日期计算支持
- 通用选项
